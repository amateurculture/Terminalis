ReadMe.txt

Easy Voice is a Unity editor extension that helps developers quickly and easily prototype in-game voice dialog
all within Unity. It uses native text to speech (TTS) to generate sound files from written dialogue lines.

Documentation can be found online at the link below:
https://game-loop.com/easy-voice/documentation/
